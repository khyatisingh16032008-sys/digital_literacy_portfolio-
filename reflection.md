## Reflection (150–200 words)

For Task 3, I explored two types of digital platforms: a coding practice platform and a cloud collaboration tool.

For Part A, I used **HackerRank**, one of the most popular platforms for practising programming and preparing for technical interviews. I completed the Python introductory challenge, which required writing a simple Python program to print "Hello, World!". While this was a beginner task, it helped me understand how HackerRank works — including code submission, test cases, and the scoring system. Over my four years of college, I plan to use HackerRank regularly to improve my coding skills and prepare for campus placements.

For Part B, I used **Google Forms**, part of the free Google Workspace suite, to create a Digital Literacy Awareness Quiz. Google Forms is a powerful collaboration tool because it allows anyone to create surveys and quizzes online, collect responses in real time, and view results automatically in a linked Google Sheet. I created five questions covering digital literacy concepts such as phishing, passwords, online presence, and email etiquette. These tools will be extremely useful for academic group work, survey-based mini-projects, and collecting peer feedback throughout college.
